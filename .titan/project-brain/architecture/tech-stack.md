# Tech Stack

## Languages
- TypeScript — 5.x (strict mode)

## Frameworks
- Next.js — 15.x (App Router, React Server Components where applicable)
- React — 19.x (client components for all audio/interactive UI)
- Tailwind CSS — v4 (CSS-first config, dark-first theming)

## Libraries
- Web Audio API — native browser API for real-time audio processing
- Zustand — lightweight state management for audio engine state
- Lucide React — SVG icon library (no emoji)
- Zod — runtime validation for user inputs and file uploads
- (To be decided by Creator/Planner) — audio visualization, export encoding

## Database
- TBD — may not be needed for initial version (client-side only)

## DevOps / Infra
- Vercel — likely deployment target (Next.js native)
- GitHub Actions — CI/CD (if needed)

## Package Manager
- pnpm (preferred) or npm

## Runtime Requirements
- Node.js >= 20
- Modern browsers with Web Audio API support (Chrome, Firefox, Safari, Edge)
