# User Preferences

## Design Preferences
- No emoji characters — always use SVG icon libraries (Lucide, Feather, Heroicons)
- Always use royalty-free stock images (Unsplash, Pexels) for visual areas
- No hardcoded colors — theme tokens only
- **Peak Studio specific:** Liquid/glassy aesthetic — frosted glass panels, backdrop-filter, translucent layers, fluid animations on dark backgrounds. This project purposefully uses glassmorphism as a design choice.
- Be creative with design: topic-relevant photography, subtle audio patterns, professional dark color combos

## Development Preferences
- High agent autonomy — decide most defaults, show results
- Expected business features must be configurable
- Build mode: solid reusable codebase (balanced pace, good structure from day one)
- Feature scope: core effects MVP first, architecture supports future expansion
- Deployment target includes Railway-like environments
- Persistence must be externalized if needed
