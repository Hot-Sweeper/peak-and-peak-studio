# User Language Translations

> Maps the user's informal or shorthand terms to their actual technical meanings.
> Check this file BEFORE asking the user for clarification — the answer might already be here.

## Audio Terms
| User says... | They mean... |
|---|---|
| "slowed" | Reduced playback rate (typically 0.8x-0.9x speed) |
| "reverb" | Convolution reverb effect using impulse responses |
| "bass boost" | Low-frequency emphasis via BiquadFilter (lowshelf) |
| "echo" | Delay effect with feedback loop |
| "pitch" | Pitch shifting (independent of speed change) |
