# Project Overview

## What Is This Project?
Peak and Peak Studio is a browser-based slowed & reverb audio effects studio. Users upload audio files and apply real-time effects (slow, reverb, bass boost, pitch shift, echo, and more) directly in the browser using the Web Audio API. Inspired by slowedandreverb.studio — built with a solid reusable codebase that can expand into a full-featured studio over time.

## Build Mode
**Solid reusable codebase** — good structure from the start, balanced pace, architected for growth.

## Design Direction
**Liquid glassy aesthetic** — dark backgrounds with frosted glass panels, translucent layers, fluid animations. Not generic SaaS — this should feel like a premium audio tool with depth and dimension. Use backdrop-filter, glass-like borders, subtle gradients on dark surfaces, and smooth micro-interactions.

## Goals
- Provide a fast, intuitive, browser-based audio effects experience
- Start with core effects, but architecture supports easy expansion to full studio feature set
- Allow users to export processed audio (WAV, MP3)
- Deliver a distinctive liquid/glassy dark-themed design
- Be responsive and functional on both desktop and mobile

## Target Users
- Music enthusiasts who want to create slowed + reverb edits
- Content creators making aesthetic audio for videos
- Casual users exploring audio effects for fun
- Producers looking for quick audio manipulation tools

## Core Features (MVP)
- Audio file upload (drag-and-drop, file picker)
- Real-time audio effects: slow, reverb, bass boost, pitch shift, speed control
- Effect parameter controls (sliders, toggles)
- Audio waveform visualization
- Audio export/download (WAV, MP3)
- Responsive dark-themed UI with liquid/glassy aesthetic

## Future Features (Post-MVP)
- Preset system for saving/loading effect chains
- Nightcore mode
- Audio trimming (start/end)
- Multiple remixes at once
- Auto-save remixes
- Custom default settings
- Pro tier with license keys
- Browser extension
